Prework Study Guide

  • HTMl stands for Hypertext Markup Language.
  • Used to build the structure of a web page.
  • HTML attributes always defined in the beginning and never at closing.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS stands for Cascading style Sheets.
  • Used to define the style of a webpage.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Layout, visual effect and background color are all examples of what can be done with CSS.

Git

  • Git is a distributed version control system.
  • Tracks changes in any set of computer files.
  • Used for coordinating work among programmers.
  • Speed data integrity, support for distributed, non-linear workflows (thousands of parallel branches running on different computers).
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript
